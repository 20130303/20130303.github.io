关于这个网站

这个静态网站是通过 GitHub Pages 托管的，使用 HTML、CSS 和 JavaScript 构建，并集成了 Tailwind CSS 框架实现现代化的设计。

GitHub Pages 提供了免费的静态网站托管服务，非常适合展示个人项目、作品集或搭建技术博客。你只需将代码推送到 GitHub 仓库，启用 Pages 功能，网站就能在互联网上访问。
